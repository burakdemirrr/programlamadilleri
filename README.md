# programlamadilleri
Programlama Dilleri Final Projesi

Projemizin arayüzü yoktur.Terminal üzerinden çalışmaktadır.

Projemizi çalıştırmak için sınıf ismimizden bir tane nesne oluşturmamız lazım.

Örneğin (x = Tez("TezDosyaAdı.docx")) şeklinde bir nesne oluşturulmalı.

Nesnemiz oluştu.Artık programın tüm fonksiyonlarını kullanabilirsiniz.
